# ACE Databricks — Production Notebooks

Production-only copies of the pipeline notebooks.
No test cells, no verification stubs, no diagnostic output.

## Prerequisites
- Unity Catalog permissions granted (run 01 first)
- Azure SP credentials stored in Databricks Secrets (`ace-secrets` scope)
- Sonnet external model registered in Model Serving (verify name below)

## Sonnet endpoint name

> **Verify before running:** Databricks Admin → Model Serving → External Models
> Expected value: `databricks-claude-sonnet-4-5` or `databricks-claude-sonnet-4-6`
> This value must match `LLM_ENDPOINT` in notebooks 03, 04, and 05 (cell 3 of each).

To update the endpoint name after verifying, edit `build_prod_notebooks.py` (in the parent folder) and re-run it — all three notebooks are regenerated in one pass.

## Execution order

| Step | Notebook | Notes |
|------|----------|-------|
| 1 | `01_infrastructure.ipynb` | Creates Delta tables + Vector Search index. Run once per environment. |
| 2 | `02_pdf_ingestion.ipynb` | Set `WIPE_BEFORE_INGEST=True` for a clean load; `False` for incremental. |
| 3 | `03_rag_agent.ipynb` | Registers + deploys RAG agent endpoint. |
| 4 | `04_page_context_agent.ipynb` | Registers + deploys page-context endpoint. |
| 5 | `05_supervisor_agent.ipynb` | Registers + deploys supervisor endpoint. Depends on 03 + 04 being live. |
| 6 | `07_serving_endpoint.ipynb` | Creates/updates `ace-chat-endpoint`. Run last. |

`06_monitoring_agent.ipynb` is not included in prod — run from the dev folder if needed.

## WIPE_BEFORE_INGEST flag (notebook 02)

| Value | Behaviour |
|-------|-----------|
| `True` | Deletes all existing chunks before ingestion. Use for full reloads or schema changes. |
| `False` | Appends/updates only. Safe for routine PDF additions. |

## Regenerating these notebooks

The notebooks in this folder are generated by `../build_prod_notebooks.py`.
Do not edit the prod notebooks directly — edits will be overwritten on the next build.
Edit the source notebooks in `../` instead, then re-run the build script.
